# Wolf Grammar Quest - Project Summary

## Overview

An educational web app to help a 10-year-old learn Year 5 UK English curriculum, specifically grammar and punctuation skills tested in NFER-style assessments.

## Target User

- 10-year-old girl
- Year 5 UK curriculum
- Struggles with grammar and punctuation in reading tests (particularly NFER style)
- Loves wolves

## App Concept

**Wolf Grammar Quest** - A calm, focused learning app where the user works through grammar challenges to build a wolf pack. No frantic timers or flashy distractions ‚Äî just satisfying progression.

### Question Format

- 10-12 NFER-style questions per territory
- All wolf-themed sentences
- Mix of question types:
  - Tap the correct word
  - Multiple choice
  - "Which sentence is correct?"
  - "What does this mean?"

### Reward System

- Starts with Luna (Alpha wolf)
- Earn a new wolf for completing each territory with 70%+ score
- Each territory has a unique wolf role: Scout, Tracker, Hunter, Guardian, Howler, Shadow, Elder, Storyteller
- User names each wolf they earn
- Each wolf comes with a real wolf fact

### Learning Support

- **Correct answers**: Brief, encouraging explanation
- **Wrong answers**: Full concept definition + explanation + correct answer (reinforces learning at the moment of need)

## Technical Details

- React single-file component
- Tailwind CSS for styling
- No external dependencies beyond React
- Designed to work as a web app on tablet

## Content Safety

- All content reviewed and child-appropriate
- All grammar answers verified for accuracy
- Wolf-themed sentences only (natural wolf behaviours)
- UK English spelling and terminology throughout
